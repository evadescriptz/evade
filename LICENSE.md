## 🛠️ Troubleshooting Common Issues

**❌ Script GUI doesn't appear after injection**
The script loaded but the HUD panel failed to render, usually caused by a resolution conflict.
**Solution:** Set `gui_position` to `"TopLeft"` in your `config.lua` and re-inject. If still missing, scale down with `gui_scale = 0.8`.

---

**❌ Auto Bhop stops working mid-round**
The **Evade Script Auto Bhop** engine disengages unexpectedly, usually after a vault or ledge grab animation.
**Solution:** Toggle `auto_bhop` off and back on via the HUD panel. A permanent fix is included in the next patch.

---

**❌ All Emotes not loading on first inject**
The emote list appears empty or only shows default emotes despite the toggle being active.
**Solution:** Wait for the full game lobby to load before injecting. Injecting during the loading transition causes the emote registry to be read before it's fully populated.

---

**❌ Lag Switch causing full disconnect**
The **Lag Switch Evade Script** module is triggering server kick logic instead of the intended brief delay.
**Solution:** Reduce `lag_duration` to `0.06` and increase `lag_cooldown` to `5.0` in your config. This keeps the effect subtle enough to avoid detection.

---

**❌ Auto Jump No Key firing too frequently**
The passive jump mode is activating even on flat ground with no obstacles, causing erratic movement.
**Solution:** This is a floor-detection threshold issue. Set `auto_jump = false` briefly to recalibrate, then re-enable it after changing floors or maps.

---

**❌ iOS injection fails after executor install**
The executor `.ipa` opens but crashes immediately when attempting to attach to Roblox.
**Solution:** Re-trust the developer certificate at `Settings > General > VPN & Device Management` and reboot the device before trying again.
